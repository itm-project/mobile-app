# mobile-app

น่าจะต้องใช้ kotlin แหละเพราะว่าทุกคนน่าจะเขียน JAVA เป็นอยู่แล้ว

กำลังหาวิธีทำเชื่อมกับ mySQL อยู่ แล้วก็น่าจะเก็บไฟล์เป็น JSON ด้วยเพราะตารางของ mySQL น่าจะไม่พอ
